## 为什么要写博客

写博客是一种极好的学习方式。当你需要向别人解释一个概念时，你会对它有更深的理解。

> 教是最好的学 —— 费曼学习法

## 技术选型

经过反复对比，最终选择了以下技术栈：

- **Next.js 15** — App Router + Server Components
- **Tailwind CSS 4** — 原子化样式，开发效率极高
- **MDX** — Markdown 增强，支持嵌入 React 组件
- **Framer Motion** — 流畅的动画体验
